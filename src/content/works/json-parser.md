Rust で JSON パーサーを作りました。

大学の講義で学んだプログラミング言語論のアウトプットの一環として、構文解析を独自実装する練習として作りました。

<https://github.com/sor4chi/json-parser>

`crates` でモノレポ管理しており、`parser` の他に `format` と `formatter-cli` があります。

詳しくはこちらの記事をご覧ください。
